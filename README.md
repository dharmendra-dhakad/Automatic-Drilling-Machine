# Automatic Drilling Machine

Developed by: Dharmendra Dhakad

## Project Overview

The Automatic Drilling Machine is an IoT and Embedded Systems based intelligent automation project developed and continuously upgraded by me since Class 8.

## Features

- Automatic Drilling Operation
- Real-Time Monitoring
- IoT Based Wireless Control
- Intelligent Automation
- Multiple Motor Control System
- Embedded Systems Integration

## Hardware Used

- Arduino UNO
- ESP8266 / NodeMCU
- Motor Drivers
- DC Motors
- Power Supply Module

## Technologies Used

- Arduino IDE
- IoT
- ESP8266 / NodeMCU
- Circuit Designing
